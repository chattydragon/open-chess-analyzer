# Roadmap

## v0.1.0 - Public foundation

- Dependency-free PGN metadata parser
- Move counting for ordinary PGN movetext
- Markdown summary output
- Sample PGN and unit tests

## v0.2.0 - Engine integration

- Add optional Stockfish/UCI support
- Evaluate positions at configurable depth
- Store evaluation swings per move
- Identify candidate mistakes and blunders

## v0.3.0 - Learning reports

- Generate Markdown and HTML game review reports
- Add beginner-friendly explanations
- Highlight key turning points
- Add opening name integration if feasible

## v0.4.0 - Import and automation

- Lichess PGN export examples
- Chess.com PGN export examples
- Batch analysis of multiple games
- GitHub Actions CI
- Docker image

## Long-term goals

- Plugin interface for analysis modules
- Local-first privacy-preserving analysis
- Optional API-based natural-language coaching summaries
- Web UI for non-technical users
